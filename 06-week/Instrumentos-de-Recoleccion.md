# Instrumentos de Recolección de Información

---

## ¿Qué son?
Herramientas para obtener datos de forma organizada. Son el **"cómo"** recoges la información.

---

## Los 8 instrumentos — para qué sirve cada uno

| Instrumento | Para qué sirve | Ejemplo rápido |
|---|---|---|
| **Cuestionario** | Recoger opiniones o datos de muchas personas a la vez sin estar presente | Encuesta por WhatsApp |
| **Entrevista** | Profundizar en las ideas o experiencias de una persona mediante el diálogo | Entrevista de ingreso al SENA |
| **Guía de observación** | Registrar comportamientos o situaciones tal como ocurren en la realidad | Evaluar a un compañero en el taller |
| **Lista de cotejo** | Verificar si algo se cumple o no, de forma rápida y objetiva | Checklist antes de abrir un negocio |
| **Escala de valoración** | Medir con qué intensidad o frecuencia ocurre algo | Estrellas en una app |
| **Ficha documental** | Organizar y resumir información extraída de fuentes escritas o digitales | Resumen de fuentes de un trabajo |
| **Test o prueba** | Evaluar el nivel de conocimiento o habilidad de una persona | Pruebas Saber |
| **Diario de campo** | Llevar un registro continuo de lo que se observa y aprende en el entorno | Bitácora de práctica empresarial |

---

## ¿Cuándo uso cada uno?

| Situación | Instrumento recomendado |
|---|---|
| Necesito preguntarle a muchas personas | Cuestionario |
| Quiero entender a fondo una opinión | Entrevista |
| Quiero ver cómo se hace algo en la práctica | Guía de observación |
| Necesito verificar pasos o requisitos | Lista de cotejo |
| Quiero saber qué tan bueno o frecuente es algo | Escala de valoración |
| Trabajo con libros, informes o páginas web | Ficha documental |
| Quiero medir conocimiento o habilidad | Test o prueba |
| Hago trabajo de campo día a día | Diario de campo |

---

## Ventajas y desventajas de cada instrumento

| Instrumento | ✅ Ventaja | ❌ Desventaja |
|---|---|---|
| **Cuestionario** | Se aplica a muchas personas a la vez | No se puede profundizar en las respuestas |
| **Entrevista** | Permite obtener información detallada | Toma mucho tiempo y depende del entrevistador |
| **Guía de observación** | Captura la realidad tal como es | El observado puede cambiar su comportamiento |
| **Lista de cotejo** | Rápida, sencilla y objetiva | Solo responde SÍ o NO, sin matices |
| **Escala de valoración** | Permite medir intensidad o frecuencia | Las personas pueden responder sin ser honestas |
| **Ficha documental** | Organiza bien la información de fuentes | Depende de que los documentos sean confiables |
| **Test o prueba** | Mide con precisión conocimientos o habilidades | Puede generar nerviosismo y afectar el resultado |
| **Diario de campo** | Registra detalles que otros instrumentos pierden | Es subjetivo y depende de quien lo escribe |
